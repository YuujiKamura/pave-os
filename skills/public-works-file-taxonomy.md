# 公共土木工事ファイルタクソノミー

## 入札時資料（発注者が提供する設計図書）

公共工事では入札時に発注者が設計図書一式を提供する。これが工事の全体像を定義する最重要資料群。

| ラベル | ファイル名パターン | 重要度 | 内容 |
|--------|-------------------|--------|------|
| `design_document` | 工事設計書 | critical | 工種・数量・単価・金額の一覧。工事の全体像の正解ソース |
| `special_spec` | 特記仕様書 | critical | この工事固有の施工条件・管理基準・使用材料指定 |
| `common_spec` | 共通仕様書 | high | 自治体の土木工事共通ルール（厚い冊子） |
| `conditions_list` | 施工条件明示 | high | 工期・施工時間帯・交通規制等の制約条件 |
| `plan_drawing` | 平面図 | high | 施工範囲の平面的な位置・寸法 |
| `location_map` | 位置図, S=1/2500 | medium | 現場の広域的な位置 |
| `bid_explanation` | 工事入札説明書 | medium | 入札手続きの説明（工事内容には直接関係しない） |
| `cost_breakdown` | 工事費内訳書, 設計金額 | high | 金額の内訳構造 |
| `public_notice` | 公告文 | medium | 入札公告（工事名・工期・概要が記載） |

### 設計図書のサフィックス慣行
発注者はファイル名末尾に記号を付けることがある:
- `k1`, `k2` → 設計金額・工事設計書（金額系）
- `s1`, `s2` → 特記仕様書・施工条件明示（仕様系）
- `z1`, `z2` → 位置図・平面図（図面系）
- `h1` → 法定福利費

## 持ち回り書類の判定

以下のラベルが付いたファイルの日付は工事期間と無関係な可能性が高い:
- `business_permit` — 建設業許可証は5年有効
- `insurance` — 保険証書は年度単位
- `retirement_fund` — 建退共証書は会社単位
- `catalog` — メーカーカタログ

これらの日付を工期推定に使ってはならない。

## 工事の全体像を把握するための優先読取順

1. **設計書** (`design_document`) — 工種・数量・金額で工事の全体像を把握
2. **特記仕様書** (`special_spec`) — この工事固有の条件・制約
3. **着工届** (`start_notice`) — 着工日の確定
4. **引渡申出書** (`completion_notice`) — 竣工日の確定
5. **契約書** (`contract`) — 契約金額・工期
6. **工程表** (`schedule`) — 実際の進捗
7. **施工計画書** (`construction_plan`) — 施工方法の計画
